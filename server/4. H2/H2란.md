# H2란?

H2DB는 자바 기반의 오픈소스 RDBMS이다.

+ 매우 빠르고, JDBC API를 지원한다
+ 인 메모리 데이터베이스
+ DB를 파일로 저장할 수 있다 -> 호스팅 서비스 이용하지 않고 프레젠테이션 가능하다
+ 용량이 작다
+ 브라우저 기반 콘솔 어플리케이션

> 설치는 [http://www.h2database.com](http://www.h2database.com/html/main.html)에서



### 영구 데이터베이스와 인 메모리 데이터베이스의 차이점

**영구 데이터베이스**는 실제 메모리에 데이터를 유지하므로 데이터베이스 서버가 반송되더라고 다시 사용 가능하다. 반면에 **인 메모리 데이터베이스**의 경우 데이터는 시스템 메모리에 저장되며 프로그램을 닫으면 데이터가 손실된다. 인 메모리 데이터베이스는 POC에 사용되며, 프로덕션 응용 프로그램에서는 권장하지 않는다.

> + POC(Proof Of Concept) :시제품 /test
> + 프로덕션 : 최종 사용자를 대상으로 하는 최종 릴리스 버전의 제품을 호스팅하도록 구성된 환경 / 최종본



영구 데이터베이스는 `Oracle`, `MySQL`, `Postgres` 등을 사용하며, H2는 일반적으로 메모리 데이터베이스로 사용한다.



**인 메모리 DB**

+ 컴퓨터가 꺼지면 모든 내용이 날라간다

+ 연속성이 없다

  > 연속성을 주는 방법이 있지만, 그 방법을 쓰려면 그냥 일반 DB 모드로 쓰며 된다.





### Spring Boot 설정

application.properties에 추가하여 H2 콘솔을 사용하게 한다.

```java
spring.h2.console.enabled=true
```

서버 실행 후 http://localhost:8080/h2-console로 접속한다.

sql 파일 생성

```java
create table member
(
 id bigint generated by default as identity,
 name varchar(255),
 primary key (id)
);
```
