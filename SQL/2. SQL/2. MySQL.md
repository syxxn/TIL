# MySQL



현재(2020년) 페이스북, 유튜브 등을 비롯한 유명한 서비스에서도 활발히 사용되고 있는 MySQL은 ORACLE 소유의 오픈소스 DBMS이다.



표준 데이터베이스 SQL을 사용하는 오픈 소스의 <u>RDBMS</u>으로 매우 빠르고 유연하다.

> Relational Database Management System : 관계형 데이터베이스 관리시스템



MySQL은 여러 DBMS 중에서도 특히 일반 사용자가 사용하기 편하다는 평가를 받고 있으며, 요구하는 컴퓨터 성능도 작은 편이라 부담이 덜하다. 게다가 IT 세계에서는 **LAMP**(Linux, Apache, MySQL, PHP/Perl/Python)라고 하는 개발 플랫폼의 조합이 이미 관용어가 됐을 정도로, 많은 개발자들이 보편적으로 사용한다.
