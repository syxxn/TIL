# SQL

> Structured Query Language



SQL은 데이터를 보다 쉽게 조작할 수 있도록 고안된 컴퓨터 언어이다.



### SQL의 특징

1. 키워드는 대소문자를 구별하지 않는다(CREATE ==create)
2. 쿼리를 이용해서 계산식의 결과도 구할 수 있다
3. 여러 문장을 한 줄에 연속으로 붙여서 실행 가능하다
4. 하나의 sql은 여러 줄로 입력 가능하다(MySQL은 문장의 끝을 ;로 구분함)
5. SQL을 입력하는 도중에 \c를 붙이면 취소할 수 있다



현재 SQL은 공인된 국제 표준이 있지만, DBMS에서 지원되는 SQL은 표준에 완벽히 준수하지 않기 때문에, SQL을 배울 때는 본인이 사용할 DBMS의 SQL을 공부하는 것이 가장 빠르다
