# SQL이란?

> Structured Query Language : 구조화된 쿼리 언어

: DBMS에게 내리는 명령



+ SQL은 **데이터를 보다 쉽게** 검색하고 추가, 삭제, 수정 같은 **조작을 할 수 있도록 고안된 컴퓨터 언어**이다

+ 관계형 데이터베이스(RDB)에서 데이터를 조작하고 쿼리(질의)하는 표준 수단이다

+ DML (Data Manipulation Language) : 데이터를 조작하기 위해 사용

  > INSERT, UPDATE, DELETE, SELECT 등

+ DDL (Data Definition Language) : 데이터베이스의 스키마를 정의하거나 조작하기 위해 사용

  > CREATE, DROP, ALTER 등

  > 스키마(Schema)
  >
  > : 데이터베이스를 구성하는 파일, 레코드, 항목의 형식과 상호 관계 전체 (데이터 구조)
  >
  > 1. 외부스키마 (서브 스키마) : 사용자나 응용 프로그래머가 직접 필요로 하는 데이터 구조
  > 2. 개념 스키마 (스키마) : 논리적 입장에서의 데이터베이스 전체 구조
  > 3. 내부 스키마 (물리적 스키마) : 기억 장치 내에 실질적으로 구성된 구조

+ DCL (Data Control Language) : 데이터를 제어함

  -> 권환을 관리하고, 데이터의 보안, 무결성을 정의함

  > GRANT, REVOKE 등
