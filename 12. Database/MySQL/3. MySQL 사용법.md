# MySQL 사용법

SQL (Structed Query Language) : 데이터베이스에서 데이터를 저장하거나 얻기 위해서 사용하는 표준화된 언어.

-> 소문자로 해도 되는데 구별하기 위해 명령어는 대문자로 해주는 것이 좋음

>  DB 켜기 `mysql -u (id) -p`  

> DB 접속(선택) `USE (DB명);`

> 테이블 목록 보기 `SHOW TABLES;`

>  DB 생성 `CREATE DATABASE (DB명);`

```
creat table user( -> 테이블 만드는거 (안에 속성)

 id varchar, (-> string)

 pw varchar

); 
```

>  varchar는 길이가 미정
>
> char는 길이 정해짐.

> MySQL 연결 끊기 `quit` || `exit`

## <수정>

> alter는 테이블 구조 변경
>
> update는 값을 변경

`ALTER TABLE (테이블명) ADD (속성명);`->속성 추가

`ALTER TABLE (테이블명) DROP (속성명);`->속성 삭제

`UPDATE (테이블명) SET (속성1)='값',(속성2)='값'...;`

`UPDATE 테이블명 SET 컬럼1=컬럼1의 값, 컬럼2=컬럼2의 값 WHERE 대상이 될 컬럼명=컬럼의 값`

`UPDATE'student' SET name='김랄라', birthday='2001-4-1' WHERE id=3;`

## <조회>

```
SELECT` `칼럼명1, 칼럼명2 
    ``[``FROM` `테이블명 ] 
    ``[``GROUP` `BY` `칼럼명] 
    ``[``ORDER` `BY` `칼럼명 [``ASC` `| ``DESC``]] 
    ``[LIMIT offset, 조회 할 행의 수]
```

데이터 검색

`SELECT * FROM (테이블명);`

`SELECT * FROM (테이블명) WHERE (조건1) AND(/OR)(조건2)...;`

## <권한 주기>

> 로컬에서 접속 가능한 사용자 추가하기
>
> **MySQL 8버전에서는 create user를 먼저 해주고 grant를 해줘야 함**
>
> `create user '사용자'@'localhost' identified by '비밀번호';`

`grant all privileges on [디비 이름].*to MYSQL[아이디]@[접속서버] identified by '[패스워드]'with grant option`

> 접속 서버 
>
> @% : 어떤 클라이언트에서든 접근 가능하다
>
> @localhost : 해당 컴퓨터에서만 접근 가능하다

`flush privileges`-> DBMS에 적용해라

> 계정 삭제
>
> `delete from mysql.user where User ='[삭제할 아이디]';`
>
> `delete from mysql.db where User ='[삭제할 아이디]';`
>
> `flush privileges;`
>
> > `drop user '사용자'@'localhost';`

계정 생성 및 권한 부여는 테이블에 직접 접근이 아닌 **create와 grant를 이용하는 것을 적극 권장**
