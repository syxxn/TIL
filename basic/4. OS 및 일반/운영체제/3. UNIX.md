# 3. UNIX

: 교육 및 연구 기관에서 즐겨 사용되는 범용 다중 사용자 방식의 <u>시분할</u> 운영체제

> 시분할 시스템 (time-sharing) : 컴퓨터 자원을 시간적으로 분할하여 사용할 수 있게 해 줌



### UNIX의 특징

1. C 언어 기반으로 제작된, 시분할 온라인 대화식 시스템

2. 확장성과 이식성이 우수

3. 멀티 태스킹 운영체제

4. 두 사람 이상의 사용자가 동시에 시스템을 사용하는 다중 사용자(Multi User) 시스템

5. 계층적인 디렉터리 구조

6. 네트워킹 및 시스템 보호 기능이 뛰어남

7. 프로그램 소스의 코드를 공개

8. 네트워크를 이용하여 시스템 자원을 공유하여 사용할 수 있도록 분산 처리 방식 지원

9. <u>CUI(Character User Interface)</u> 환경 제공

   > CUI (문자(콘솔) 사용자 인터페이스)
   >
   > : 컴퓨터와 사용자가 글자로 명령을 주고받는 작동방식



### UNIX의 구성

+ 커널(kernel) : UNIX의 가장 핵심. 항상 주기억 장치에 상주하여 시스템의 자원을 관리함

  > 프로세스 관리, 입출력 관리, 파일 관리, 메모리 관리, 시스템 호출, 프로세스 간 통신 관리

+ 셸(shell) : 사용자와 UNIX 간의 인터페이스 역할

  + shell 프로그램 언어를 제공하는 명령어 해석기
  + 단말기를 통해 명령을 입력받은 후 커널이나 다른 유틸리티 프로그램을 실행시켜 해당 명령 수행

+ 유틸리티(utility) : <u>DOS</u>의 외부 명령어에 해당

  > DOS (dist operating system)
  >
  > : 디스크 운영 체제의 일종으로서 디스크에 읽고 쓰기 등의 명령을 수행하는 프로그램

  + UNIX 시스템을 효과적으로 사용할 수 있도록 도와주는 응용 프로그래밍
