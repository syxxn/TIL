# 3계층 _네트워크 계층(Network layer)

: 데이터를 연결하는 다른 네트워크를 통해 전달함으로써 인터넷이 가능하게 만드는 계층

이 계층에서 가장 중요한 기능은 **데이터를 목적지까지 가장 안전하고 빠르게 전달하는 기능(라우팅)**이다

네트워크 계층은 경로를 선택하고 주소(IP)를 정하고, 경로에 따라 패킷을 전달해준다

대표적인 장비는 라우터이며,논리적인 주소 구조(IP), 곧 네트워크 관리자가 직접 주소를 할당하는 구조를 가진다



+ 역할

  + 오류 제어
  + 흐름 제어

  + 라우팅

    > 네트워크 안에서 통신 데이터를 보낼 때 **최적의 경로를 선택**하는 과정

  + 세그멘테이션

    > 가살 메모리를 서로 크기가 다른 논리적 단위인 세그먼트로 분할하고 **메모리를 할당**하는 기법

  + 인터네트워킹

    > '네트워크들의 네트워크'
    >
    > 두 개 이상의 네트워크를 연결하여 네트워크 간 하드웨어나 소프트웨어를 **모두 연결**시키는 방법



+ 프로토콜

  + IP(Internet Protocol)
    + 패킷 교환 네트워크에서 정보를 주고받는데 사용하는 정보 위주의 규약
    + 호스트의 주소지정과 패킷 분할 및 조립 기능 담당
  + ICMP(Internet Control Message Protocol)
    + TCP/IP에서 IP 패킷을 처리할 때 발생되는 문제(오류) 보고
    + 진단 등과 같이 IP계층에서 필요한 기타 기능들을 수행하기 위해 사용되는 프로토콜

  + IGMP(Internet Group Management Protocol)
    + IP 멀티캐스트를 실현하기 위한 통신 프로토콜
    + PC가 멀티캐스트로 통신할 수 있다는 것을 라우터에 통지하는 규약
