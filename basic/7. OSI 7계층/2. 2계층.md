# 2계층 _데이터링크 계층(DataLink Layer)

: 물리적 계층을 통한 데이터 전송에 신뢰성을 제공

> **에러검출, 재전송, 흐름제어**



물리 계층에서 발생할 수 있는 오류를 찾아내고, 수정하는 데 필요한 기능적, 절차적 수단을 제공한다

주소 값은 물리적으로 할당 받는데, 이는 네트워크 카드가 만들어질 때부터 <u>맥 주소(MAC address)</u>가 정해져 있다는 뜻이다

> MAC 주소
>
> : 네트워크 상에서 서로를 **구분하기 위하여** 디바이스마다 할당된 물리적인 주소
>
> 인터넷에 연결되는 모든 기기에 설정되어 있으며, 그 값이 고정되어 있다- > **개별 단말기 식별** 가능



직접 이어진 곳에만 연결할 수 있으며, 물리계층을 통해 송수신되는 정보의 오류와 흐름을 관리하여 안전한 **정보 전달을 수행**할 수 있도록 도와준다

전송 단위를 **프레임**이라고 하며, 대표적인 장비로는 이더넷, 브리지, 스위치 등이 있다 (여기서 MAC 주소를 사용)



+ 프로토콜

  + Ethernet

    : 비연결성 모드, 전송속도 100Mbps 이상, LAN 구현 방식
