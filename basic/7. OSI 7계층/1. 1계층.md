# 1계층 _물리계층(Physical layer)

: 실제 장치들을 연결하기 위해 필요한 전기적, 물리적 세부 사항들을 정의한다

> 하드웨어적인 부분이 많아 소프트웨어와는 가장 거리가 멀다



1계층은 **7~2계층을 거쳐 <u>캡슐화</u>가 완료된 데이터를 전기 신호로 바꾸어 전송하는 역할을 한다 **

> 캡슐화 : 송신 데이터에 필요한 정보(헤더)를 붙여서 다음 게층에 보내는 기술
>
> 상위-> 캡슐화-> 하위
>
> +역캡슐화 : 헤더를 제거하는 것

물리계층은 네트워크상에서 데이터 비트를 전송하는 계층이다. 데이터 링크 개체 간의 비트 전송을 위한 물리적 연결을 설정, 유지, 해제하기 위한 수단을 제공한다.

전송 단위는 **비트(bit)**이며, 데이터를 2진수의 전기 신호로 바꾸어 전송시킨다

대표적인 장비로는 랜선, 광케이블, 모뎀, 허브, 라피터 등이 있다
