# 2. Windows

: 마이크로소프트가 개발한 컴퓨터 운영체제

> 애플이 처음으로 도입한 GUI 운영 체제인 맥 OS에 대항하여 출시됨
>
> MS-DOS에서 멀티태스킹과 GUI 환경을 제공하기 위한 응용 프로그램으로 처음 출시

+ 전 세계 90%의 개인용 컴퓨터에서 사용
+ 서버용 운영체제로도 영역을 넓혀가고 있음

+ 큰 시장 점유율을 차지하고 있어서, 일반 사용자들에게 매우 익숙함

+  호환되는 유명한 응용 프로그램이 많음    

  > -> 그만큼 보안 문제 취약



#### 윈도우 특징

+ GUI(그래픽 사용자 인터페이스)

  > 키보드로 명령어를 직접 입력하지 않고, 마우스로 아이콘이나 메뉴를 선택하여 모든 작업 수행

+ Preemptive Multi-Tasking(선점형 멀티태스킹)

  > 멀티태스킹(멀티 프로그래밍)을 하면서 운영 체제가 각 작업의 CPU 이용시간을 제어하여 응용프로그램 실행 중 문제가 발생하면 해당 프로그램을 강제 종료시키고 모든 시스템 자원을 반환하는 방식

+ OLE(Object Linking and Embedding) 사용

  > 다른 여러 응용 프로그램에서 작성된 문자나 그림 등의 개체(Object)를 현재 작성중인 문서에 자유롭게 연결하거나 삽입하여 편집할 수 있게 하는 기능

+ Single-User 시스템

  > 컴퓨터 한 대를 한 사람만이 독점해서 사용



 ##### 윈도우 1.0부터 3.1x에 이르는 버전들은 운영 체제가 아닌, MS-DOS 하에서 동작하는 일종의 소프트웨어로 간주
